#Включение подсветки синтаксиса языка ассемблера NeuroMatrix в [Notepad++](https://notepad-plus-plus.org/)

## Установка в Notepad++  
[Главное меню->Синтаксисы->Задать свой синтаксис]  
[Импортир..] ... выбрать NeuroMatrix-Notepad-UDL.xml  
[Польз. язык]  ... выбрать "NeuroMartix Assembly"  
[Сохранить как...] ... [Name:]  ввести NeuroMatrix ASM  


в меню "Главное меню->Синтаксисы" должно появиться "NeuroMatrix ASM"  

P.S. Если подсветка при открытии .asm не включается автоматически необходимо обновить версию Notepad++, либо выбрать  "NeuroMatrix ASM" вручную.
